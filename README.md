# Java Web Chat Application

A web-based real-time chat application developed using Java technologies.

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- JavaScript

## Features
- User login system
- Real-time chat messaging
- Message storage using MySQL database
- Simple and responsive web interface

## Tools Used
- Apache Tomcat Server
- MySQL Database
- Eclipse IDE
